#  コンピュータ科学実験3

コンピュータ科学実験3のソースコード管理用リポジトリです



## Usage

### Setup

```bash
$ python -m venv env_name
$ source env_name/bin/activate
$ pip install -r requirements.txt
```



### parsing

```bash
$ cd kadai2
$ sh parse.sh path/to/test/code
```



### scan

```bash
$ cd kadai1
$ python scanner.py path/to/test/code
```
